# SkillForge
A personalized learning feed for developers.

## Features
* User profiles with selected technologies
* Dashboard with latest items filtered by user's tech stack
* Item detail view with summary and external links
* User can modify tech stack preferences at any time

## Usage
1. Create a user with a tech stack
2. Add items with tags and links
3. Get the dashboard for a user
4. Get an item by ID
5. Update a user's tech stack
